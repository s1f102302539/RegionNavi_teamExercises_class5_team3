### セットアップ

1. Python の仮想環境を作成し、有効化：

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

2. サーバー設営
npm run dev
ローカル：http://localhost:3000
